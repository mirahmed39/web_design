Data Description: The dataset I have used in this assignment represents records of students that received TAP (tuition assistance program for NY state eligible residents). The records include the TAP award recipients and “dollar amounts by income, age group, and more” headcount within each income level group and more since the beginning of the year 2000. I think this dataset is credible since it is a large file that contains a big sum of records. Moreover, this dataset has been published by the state of New York and maintained by “Open Data NY” which is “Governor Cuomo’s award-winning initiative of policies, programs, and tools that provide public access to digital data for collaboration and analysis”.
