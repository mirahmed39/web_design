Movies Application with LAMP Stack

Description: This project allows the user the user to see movie inforamtion based on their category and country of origin. Upon selecting one option from each dropbox, the user will be able to see relevant information provided by the database in a tabular fashion. The table would be empty if the database cannot find results to the correspoding user options. For example, if the user selects USA as the country and Action as a cateogry, then the table will display the name of the movies relevant to that category, country of origin, category, the year they were released, and links that take to their corresponding IMDB pages.
